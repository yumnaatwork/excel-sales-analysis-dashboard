# Online Sales Data Analysis & Dashboard

Comprehensive data cleaning and exploratory data analysis (EDA) of online retail sales using Microsoft Excel.

## Project Overview

Analysis of 535,302 transaction records from an online retail business operating across 39 countries (Dec 2010 - Dec 2011). This project demonstrates end-to-end data analysis skills including data cleaning, transformation, visualization, and business insights generation.

## Tools & Techniques

### Excel Features Used:
- **Data Cleaning:** Removed 1,174 problematic records, handled missing values, flagged promotional items
- **Calculated Fields:** Revenue, time-based metrics (Year, Month, Hour), categorization
- **Pivot Tables:** 7+ pivot tables for multi-dimensional analysis
- **Data Visualization:** Interactive charts (line, bar, pie, column)
- **Slicers:** Dynamic filtering by Year, Country, Month, Day of Week
- **Dashboard Design:** Executive summary with KPIs and key insights

# Analysis Summary

## Methodology

### 1. Data Cleaning (Phase 1-2)
- Removed duplicate/invalid records
- Handled missing values
- Created flag variables
- Validated data integrity

### 2. Feature Engineering (Phase 2)
- Created time-based features (Year, Month, Hour)
- Calculated Revenue (Quantity × Price)
- Categorized orders by size
- Flagged special transactions

### 3. Exploratory Analysis (Phase 3-4)
- Created 7 pivot tables
- Built interactive visualizations
- Calculated summary statistics
- Identified trends and patterns

### 4. Dashboard Creation (Phase 5)
- Designed executive summary
- Created Useful Visualiations
- Documented key insights
- Created professional layout

## Key Recommendations

1. **Diversify Geographic Revenue**
   - Risk: 82% dependency on UK market
   - Action: Invest in marketing for top 5 non-UK countries

2. **Convert Guest Customers**
   - Opportunity: 17% revenue from guests
   - Action: Incentivize account creation (discount codes, loyalty points)

3. **Optimize Staffing**
   - Finding: 10am-3pm peak hours
   - Action: Ensure maximum support coverage during peak times

4. **Focus on Top Products**
   - Finding: Top 20 products drive significant revenue
   - Action: Ensure stock availability, promote heavily
